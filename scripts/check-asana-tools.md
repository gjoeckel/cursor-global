# Asana MCP Tools Discovery

**After restarting Cursor, follow these steps to document Asana tools:**

---

## Steps to Check Asana Tools

1. **Open Cursor Settings**
   - Press `Cmd+,` (or Cursor → Settings)
   - Or: File → Preferences → Settings

2. **Navigate to MCP**
   - Search for "MCP" in settings
   - Or go to: Developer → MCP Tools

3. **Find Asana Server**
   - Look for "asana" in the list of MCP servers
   - Click on it to expand

4. **List All Tools**
   - You should see a list of available tools
   - Each tool will have:
     - Tool name
     - Description
     - Parameters (if visible)
